# WDD131 - Week 01: HTML and CSS Layout

## Author
Martha Sibiya

## Repository Overview
This repository contains exercises for WDD131 (Web Design and Development).  
Week 01 focuses on reviewing **HTML and CSS layout principles**, including the use of **semantic HTML**, **CSS Grid**, and **Flexbox**.

## Week 01 Layout
- `week01/` folder contains:
  - `basic-layout.html` → The main HTML file for Week 01
  - `styles/layout.css` → External CSS for the layout

## Layout Features
- **Header** with navigation menu (Flexbox)  
- **Sidebar** and **Main content area** (CSS Grid)  
- **Footer**  
- Placeholder content included for demonstration purposes
